Tareas Bany

Tareas Bany es una aplicación web diseñada para la gestión sencilla y efectiva de tareas pendientes. Permite a los usuarios crear, completar y eliminar tareas de manera rápida, ofreciendo una experiencia visual limpia y agradable que facilita la organización diaria.

Características principales:
- Creación instantánea de nuevas tareas con un solo clic o presionando Enter
- Marcado de tareas como completadas
- Eliminación individual de tareas
- Diseño de paleta de colores suaves que reduce la fatiga visual

Tecnologías utilizadas
- HTML5 Estructura semántica y accesible
- CSS3 Flexbox y Grid para el layout, transiciones suaves y diseño moderno con sombras sutiles
- JavaScript puro Lógica interactiva, manejo del DOM y almacenamiento local sin dependencias externas

Historia de usuario 1
Como usuario de Tareas Bany, quiero una interfaz sencilla y limpia para poder agregar tareas nuevas con un solo clic o presionando Enter, sin complicaciones ni pasos innecesarios.
Historia de usuario 2
Como usuario de Tareas Bany, quiero marcar una tarea como completada y que automáticamente se mueva a la columna “Completadas” con un efecto visual suave, para sentir satisfacción y ver claramente mi progreso diario.
Historia de usuario 3
Como usuario de Tareas Bany, quiero poder eliminar cualquier tarea que ya no necesite con un solo clic, para mantener mi lista siempre ordenada y sin elementos obsoletos.
Historia de usuario 4
Como usuario de Tareas Bany, quiero que todas mis tareas se guarden automáticamente en el navegador aunque cierre la pestaña o reinicie el equipo, para nunca perder la información que ya organicé.
Historia de usuario 5
Como usuario de Tareas Bany, quiero un diseño con colores suaves para poder gestionar mis tareas en cualquier momento y lugar sin cansar la vista.
